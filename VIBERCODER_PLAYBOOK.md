## 🎮 核心快捷键 (Cheat Sheet)

### 1. 窗口导航 (无缝穿梭)
> 无视 Vim 与 Tmux 的边界，像在同一个应用里一样移动。

| 快捷键 | 动作 |
| :--- | :--- |
| **`Ctrl + h`** | 向**左**跳 (Vim Window -> Tmux Pane) |
| **`Ctrl + j`** | 向**下**跳 |
| **`Ctrl + k`** | 向**上**跳 |
| **`Ctrl + l`** | 向**右**跳 (代码 -> 终端) |

### 2. 布局调整
*   **微调 (Vim 内)**: `Alt + h/j/k/l` (调整代码分割窗口大小)
*   **宏调 (Tmux)**: `Ctrl+b` -> 按住 `Shift` + `H/J/K/L` (调整面板比例)
*   **专注模式**: `Ctrl+b` -> `z` (当前面板最大化/恢复)

### 3. Vim 常用操作
*   **找文件**: `<Space><Space>` (Files)
*   **搜内容**: `<Space>/` (Grep)
*   **文件树**: `<Space>e` (Explorer)
*   **Git**: `<Space>gg` (Lazygit)

### 4. Tmux 进阶 (高阶多任务)
> 像操作系统一样管理你的开发环境。

| 场景 | 快捷键 (前缀: `Ctrl+b`) | 说明 |
| :--- | :--- | :--- |
| **新建窗口** | `Ctrl+b` -> `c` | (Create) 开一个新窗口跑测试或看日志。 |
| **切换窗口** | `Ctrl+b` -> `w` | **[推荐]** 弹出列表，按 `j/k` 选择窗口。 |
| **重命名** | `Ctrl+b` -> `,` | 给当前窗口起个名 (如 "server")。 |
| **项目切换** | `Ctrl+b` -> `s` | **[神器]** 弹出所有 Session 列表，在不同项目间秒切。 |
| **热重载** | `Ctrl+b` -> `:source ~/.tmux.conf` | 修改配置后立即生效。 |

## 🤖 AI 协同工作流

### 场景 A: 代码补全 (Copilot)
*   打字时出现灰色建议 -> **`Alt + l`** 接受。
*   **首次使用**: 运行 `:Copilot auth` 登录。

### 场景 B: 智能对话 (Avante)
*   **召唤**: 选中代码或空行 -> **`<Space>aa`**。
*   **指令**: "解释这段代码" 或 "重构这个函数"。
*   **切换模型**: `<Space>ap` (在 DeepSeek / Gemini 间切换)。

### 场景 C: 全局生成 (Aider)
*   跳到终端 (`Ctrl+l`) -> 运行 `aider`。
*   适合跨文件的复杂任务。
